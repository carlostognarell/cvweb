CUBO VENTAS

Este proyecto consiste en la creación de un Cubo de Datos que alimenta la reportería respecto a las ventas históricas de la compañía.
Para la extracción de información se usaron consultas complejas en SQL Server y MySQL; las que permiten mediante una ETL cargar la información al Datawarehouse.
La información histórica en el Datawarehouse son procesadas por un Cubo OLAP desarrollado en Analysis Services; las que posteriormente puede ser representada en distintos clientes de visualización como Excel o Power BI.

Tecnologías

  • SQL Server
  • MySQL
  • SQL Server Integration Services
  • SQL Server Analysis Services
Omnix

Un OMS (Order Management System) es un software que permite la administración de las ventas desde su llegada al sistema de la compañía hasta su entrega; manejando cambios de estados, stock y traspasos de mercadería.
Este proyecto consiste en la elaboración de una librería de APIs para la comunicación entre el ERP empresarial y las distintas funcionalidades del OMS.

Tecnologías

  • Azure Api Management
  • SQL Server
  • Entity Framework
  • .NET Core
POS

Proyecto que consiste en el cambio de los Puntos de Ventas de las tiendas físicas de la compañía; para lo cuboModal se compraron e instalaron nuevos equipos (redes, computadores para cajas y servidores para tiendas).
Mi labor consistió en desarrollar y administrar la librería de APIs que sirven para la comunicación entre los datos del ERP de la empresa y el software de las nuevas cajas integrando ventas, clientes e inventario.

  • Azure Api Management
  • SQL Server
  • Entity Framework
  • .NET Core
GUARDARROPÍA WALMART

Aplicación que se utiliza para el enrolamiento de nuevos usuarios y su administrtación de uniformes.
Este proyecto permite enrolar la huella digital mediante una aplicación desarrollada en WinfowsForm conectada a un huellero digital; también permite la entrega y recepción de prendas autenticando al usuario con su huella.
Este proyecto también cuenta con una aplicación de consola programada todas las noches para sincronizar información desde y hacia el ERP.
